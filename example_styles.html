A small collection of css files based off of Skeleton 1that encourages visual consistency and rapid front-end development.

How does it work?

Frame provides a handful of fun classes that you can apply to your html on the fly. The CSS is divided into the following sections:

  1. Grid.css - A responsive grid that scales from 320px to 1440px. 2
  2. Base.css - Reset, typography, and basic site elements.
  3. Elements.css - Helper classes so you can fiddle less and make more.
  4. Layout.css - This is where you code.

Getting Started

  1. Divide up your site into macro sections which you will assign the class ".container"
  2. Assign sections in your containers a column value (16 columns max).
  3. Use the classes as a starting point. Overwrite as needed.

Typography

The typography of Frame is pretty straightforward. I'm not a huge fan (don't hate) of Helvetica. I would have gone with Proxima Nova but didn't want to pay. So i'm using Roboto as an approximation.

Nested Columns

Leftmost Nested Columns

Nested columns have an additional class added which removes the left or right margin of a column. The leftmost nested column usually requires the ".front" class to remove the left margin.

Middle Nested Columns

Columns nested in the middle don't need the additional nested column class

Rightmost Nested Columns

The last nested column in a row has the additional class ".back" which removes the right margin.
